I am a generalist developer with an emphasis on the web.

Major accomplishments

  • designed and built a web product prototype responsible for a major angel investment (a 10x ROI on services)
  • conceived and built an online digital narrative that won Microsoft's Imagine Cup against thousands of international competitors
  • built a prototype interactive online training course for the World Bank's IFC, to be piloted across Southeast Asia

Further details about each of these projects (and more) can be found on my portfolio page.

A Unique Background

I have been building front-ends and web applications for startups, research labs, and large institutions for the last five years.

Before that, I trained in competitive algorithm programming (World Finalist in the 2008 ACM ICPC), which gives me insight into the back-end functionality of web apps as well as very sharp coding skills under time pressure.

If it sounds like I could help your next big idea, please get in touch.
