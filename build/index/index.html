software product development

Hi, I'm Marc Paré and I design and write software, particularly as part of the product development process. For example, I have

  • designed and built a web software product prototype that raised a $500k angel investment
  • conceived and built an interactive digital narrative that won Microsoft's Imagine Cup against thousands of international competitors
  • built an interactive training course for the World Bank's IFC

I have done consulting and contract work around the world. I have a generalist skill set, capable on the "full stack" of web technologies. I particularly enjoy work with user interfaces and APIs (which are a sort of user interface for software developers).

My background in software is writing software to implement and support mechanical engineering simulation. I also trained in competitive algorithm programming, reaching the World Finals in the ACM ICPC in 2008.

If it sounds like I could help your business, please get in touch. I work primarily in Python, PHP, and Javascript these days.
